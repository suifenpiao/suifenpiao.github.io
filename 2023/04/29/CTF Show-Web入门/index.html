信息收集

一般收集

前几道题都是常规的信息收集和路径扫描工具可以直接扫出的文件，具体有

  • 源码泄露
    • f12查看源码
  • .git/.svn泄露
    • git/svn hack利用工具，题目中直接访问即可获得flag
  • 备份文件泄露
    • dirsearch扫描敏感路径查找&特征框架/cms 备份文件路径查找
  • robots.txt
    • 网站反爬虫txt文件访问即得
  • vim异常退出泄露
    • 异常退出导致生成当前编辑文件.后缀.swp文件如index.php.swp
  • 网络泄露
    • 返回包信息泄露，服务器/操作系统/语言等
  • cookie/session泄露

社工收集

  • 在目标网站需要登录时我们可以提前阅览整个网站，是否留下邮箱个人 信息等，进行登录等操作
  • 目标网站是否泄露文档，我们可以查看文档进一步获取有用信息
  • DNS泄露，查看当前IP的历史解析记录
  • 浏览器特定语法寻找有关网站的二级域名等信息

其他泄露

  • 探针泄露
    • phpmyadmin
    • 雅黑探针
    • 什么是php探针
      • php探针是用来探测空间、服务器运行状况和PHP信息用的，探针可以实时查看服务器硬盘资源、内存占用、网卡流量、系统负载、服务器时间等信息
    • 使用后若未删除,则可能造成信息泄露
  • sql备份文件泄露
    • backup.sql
  • 密钥泄露
    • 加密算法的密钥偏移量等信息前端保存,导致可以逆向出密码明文
  • js逻辑泄露
    • 通过审计js代码达成目的

爆破

web21

进入题目环境直接一个登录框一个密码输入框

输入用户名admin密码随便输入,bp抓包,发现字段Authorization: Basic YWRtaW46MjIzNDU=

对该字段后面部分send to decoder进行base64解码：admin:xxxx

思路

  1. 密码前面拼接admin:
  2. 对admin:xxx进行base64编码
  3. 根据题目提示,等号可能影响结果,删除所有等号
  4. 取消url编码,防止admin:中的:被编码影响结果

显然需要对xxxx进行爆破,两种方式

PHP特性

89

intval()函数

利用数组绕过第一个if,同时当数组有值时,intval函数返回1,没有时返回0,据此通过第二个if,构造payload:?num[]=1,获得flag

90

根据intval函数特性,第二个参数为0时,传入各个进制特定格式再传入值可自动解析,因此传入4476的十六进制值0x117c即可获得flag
